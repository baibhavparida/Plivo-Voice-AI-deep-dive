## Learning Paths by Role

Different roles require different knowledge areas. Use these curated paths to focus your learning.

### Developer Learning Path

For software engineers building voice AI applications, this path provides the technical foundation and implementation skills you need.

**Stage 1: Core Concepts (Week 1)**

Start with understanding the fundamental technologies:

1. [Introduction to Voice AI](/topics/welcome/introduction) - Establish context
2. [Speech Recognition Overview](/topics/foundations/speech-recognition/overview) - Understand STT

5. [Neural TTS](/topics/foundations/speech-synthesis/neural-tts) - Modern TTS architectures

**Stage 2: Infrastructure (Week 2)**

Learn the systems that support voice AI:

1. [Audio Pipeline Overview](/topics/infrastructure/audio-pipeline/overview) - End-to-end audio
2. [Voice Activity Detection](/topics/infrastructure/audio-pipeline/vad) - Detecting speech
3. [Noise Cancellation](/topics/infrastructure/audio-pipeline/noise-cancellation) - Audio quality
4. [Streaming Overview](/topics/infrastructure/streaming/overview) - Real-time processing
5. [Latency Optimization](/topics/infrastructure/streaming/latency-optimization) - Performance tuning

**Stage 3: Integration (Week 3)**

Connect components into working systems:

3. [Function Calling](/topics/llm-integration/function-calling) - Enabling agent actions

**Stage 4: Architecture (Week 4)**

## Quick Reference

### Glossary of Key Terms

| Term | Definition |
|------|------------|
| **STT** | Speech-to-Text, converting audio to text |
| **TTS** | Text-to-Speech, converting text to audio |
| **ASR** | Automatic Speech Recognition (synonym for STT) |
| **NLU** | Natural Language Understanding |
| **NLG** | Natural Language Generation |
| **LLM** | Large Language Model |
| **VAD** | Voice Activity Detection |
| **WER** | Word Error Rate (STT accuracy metric) |
| **TTFB** | Time to First Byte |
| **RTF** | Real-Time Factor |
| **S2S** | Speech-to-Speech |

### Key Metrics Reference

| Metric | Good | Excellent | Context |
|--------|------|-----------|---------|
| STT WER | < 10% | < 5% | Clean audio conditions |
| End-to-end latency | < 1000ms | < 500ms | Conversational applications |
| TTS TTFB | < 200ms | < 100ms | Streaming synthesis |
| VAD accuracy | > 95% | > 99% | Speech detection |
