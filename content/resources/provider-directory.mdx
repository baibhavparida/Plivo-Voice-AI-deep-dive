## Selection Guidance

### By Use Case

| Use Case | STT | LLM | TTS | Orchestration | Telephony |
|----------|-----|-----|-----|---------------|-----------|
| **Customer Service** | Deepgram | GPT-4o-mini | Cartesia | Vapi | Plivo |
| **Sales Calls** | Deepgram | Claude Haiku | ElevenLabs | Retell | Twilio |
| **Healthcare** | Deepgram Medical | GPT-4o | Azure | Custom | Bandwidth |
| **Multilingual** | Google Chirp | Gemini Flash | Google TTS | Vocode | Vonage |
| **Cost-Sensitive** | AssemblyAI | Groq Llama | OpenAI tts-1 | Pipecat | Telnyx |
| **Maximum Quality** | Deepgram Nova-3 | Claude Sonnet | ElevenLabs v3 | LiveKit | Twilio |

### By Priority

**Lowest Latency Stack**:
- STT: Deepgram Nova-3
- LLM: Groq Llama 3.1 70B
- TTS: Cartesia Sonic
- Expected E2E: ~450ms

**Best Quality Stack**:
- STT: Deepgram Nova-3
- LLM: Claude 3.5 Sonnet
- TTS: ElevenLabs Multilingual v2
- Expected E2E: ~1400ms

**Most Cost-Effective Stack**:
- STT: AssemblyAI
- LLM: GPT-4o-mini
- TTS: OpenAI tts-1
- Expected cost: ~$0.002/min LLM + provider costs

**Best Multilingual Stack**:
- STT: Google Chirp 3
- LLM: Gemini 2.0 Flash
- TTS: Google Cloud TTS
- Languages: 125+

### Enterprise Considerations

| Factor | Recommended Approach |
|--------|---------------------|
| **Data Privacy** | On-prem options (Deepgram, Azure) |
| **Compliance** | HIPAA: Deepgram Medical, Azure |
| **SLA Requirements** | Enterprise tiers from major providers |
| **Support** | Twilio, Azure, AWS for 24/7 support |
| **Vendor Lock-in** | Open-source (Vocode, Pipecat) for flexibility |
